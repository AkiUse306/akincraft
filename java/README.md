# Java Akincraft Prototype

This folder contains the Java + LWJGL + OpenGL implementation of Akincraft.

## Requirements

- Java 21 or newer
- Gradle wrapper included

## Build and Run

```bash
cd java
./gradlew run
```

The project uses LWJGL for windowing, input, and OpenGL rendering. It demonstrates procedural chunk generation, greedy mesh generation, first-person controls, and biome-driven terrain.
